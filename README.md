# Overview:
Docker for ROS2 spacemouse node.

# 1. Installation & config:
> git clone https://github.com/bkuster0/spacemouse-ros2-docker.git

> cd spacemouse-ros2-docker/docker

> nano docker-compose.yml

Modify the ROS_DOMAIN_ID and ROS_LOCALHOST_ONLY according to your preferences.

# 2. Run
> cd spacemouse-ros2-docker/docker

> docker compose up
